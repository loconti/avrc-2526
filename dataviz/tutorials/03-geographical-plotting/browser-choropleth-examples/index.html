Browser Choropleth Implementations

These examples reproduce the interactive London crime-score choropleth from the notebook with three different browser mapping libraries. The goal is not to change the data story, but to show how the same choropleth logic is expressed through different APIs.

Shared analytical choices
  • Dataset: London IMD MSOA polygons
  • Variable: crime
  • CRS for browser rendering: EPSG:4326
  • Classification: Quantiles(k=5), precomputed once in Python
  • Appearance: 5-class YlOrRd palette, thin outline, neutral basemap

Leaflet

Leaflet is the clearest starting point if you want to understand an interactive choropleth as a raster basemap plus a GeoJSON overlay.

Open Leaflet Example
  • The map is built from L.tileLayer and L.geoJSON.
  • Styling happens feature by feature through a simple JavaScript function.
  • Hover and click behavior are attached inside onEachFeature.
  • The legend is a custom Leaflet control, so the class labels stay explicit.
Leaflet Home Quick Start GeoJSON Reference

deck.gl

deck.gl is useful when you want to think in terms of rendering layers: one layer for the basemap tiles and one layer for the choropleth polygons.

Open deck.gl Example
  • The basemap is a TileLayer rendered through BitmapLayer.
  • The choropleth is a GeoJsonLayer styled from the shared class index.
  • Tooltips are handled by deck.gl’s picking system instead of DOM events on features.
  • Click opens a modal with the same detail fields used in the other examples.
  • The example is useful for understanding how deck.gl separates data, view state, and layers.

MapLibre

MapLibre shows the GL-style workflow without requiring a proprietary token. This is the best example if you want styled layers and vector-like map composition with an open-source library.

Open MapLibre Example
  • The map uses a style object with raster tiles plus custom fill and line layers.
  • The choropleth is added as one GeoJSON source and two layers: fill and outline.
  • Hover uses feature state, which is a common GL-map pattern.
  • This version is close in structure to Mapbox GL JS, but remains token-free.

Open these files through a local web server, not through file://, because each example fetches shared data from the data/ directory. The preparation script prepare_london_crime_choropleth.py is the single source of truth for the exported GeoJSON, class breaks, labels, and palette metadata.
