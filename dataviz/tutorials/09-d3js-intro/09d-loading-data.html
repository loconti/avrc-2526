09d — Loading Data and D3 Helpers

Bringing real data into a D3 page: how d3.csv parses a file, why an accessor is usually needed, and the helpers D3 ships for summarising and formatting data (d3.extent, d3.group, d3.rollup, d3.format) before plotting.

1. Loading a CSV

The simplest call:

const raw = await d3.csv("./datasets/gapminder.csv");

2. The accessor pattern

d3.csv accepts an optional second argument: a function called for each row. Returning a freshly built object is the right place to coerce types:

const data = await d3.csv("./datasets/gapminder.csv", d => ({
    country:    d.country,
    continent:  d.continent,
    year:      +d.year,        // unary + coerces to a number
    lifeExp:   +d.lifeExp,
    pop:       +d.pop,
    gdpPercap: +d.gdpPercap
}));

The unary + turns the string into a number. Without it, a comparison like d.year < 2000 compares strings lexicographically — true for "1952" and false for "2007" for the wrong reason. After coercion, the same row looks like:

3. Summary statistics with D3 helpers

Once the data is typed correctly, D3 provides small helpers for the kinds of summaries every chart needs:

d3.min(data, d => d.lifeExp);     // smallest
d3.max(data, d => d.lifeExp);     // largest
d3.extent(data, d => d.lifeExp);  // [min, max]  — handy for scale .domain()
d3.mean(data, d => d.lifeExp);    // arithmetic mean
d3.median(data, d => d.lifeExp);
d3.sum(data, d => d.pop);

4. Grouping and rolling up

Two related helpers turn an array of rows into a Map keyed by some field:

  • d3.group(data, d => d.continent) — groups rows by continent (returns Map<continent, row[]>)
  • d3.rollup(data, reducer, d => d.continent) — applies a reducer to each group (returns Map<continent, summary>)

Rolling up the mean life expectancy per continent in 2007:

const meanByContinent = d3.rollup(
    data2007,
    rows => d3.mean(rows, d => d.lifeExp),
    d => d.continent
);
// → Map { "Africa" => 54.81, "Americas" => 73.61, ... }

5. Formatting numbers and dates

Raw numbers are rarely the right thing to put on a chart. D3 ships compact format specifiers:

d3.format(",")(1234567);          // "1,234,567"
d3.format("$,.0f")(1234567);      // "$1,234,567"
d3.format(".1%")(0.0734);         // "7.3%"
d3.format("~s")(2400);            // "2.4k"

const fmtDate = d3.timeFormat("%Y-%m-%d");
fmtDate(new Date());              // e.g. "2026-04-28"
