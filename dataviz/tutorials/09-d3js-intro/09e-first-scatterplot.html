09e — First Scatterplot

Gapminder scatterplot, end-to-end: GDP per capita on a log axis, life expectancy on the y-axis, size by population, color by continent. Selections, scales, axes, data join, legend, and tooltips, in about 70 lines of D3.

The finished chart

Step 1 — the margin convention

Same skeleton as every previous tutorial. The right margin is wider than usual to leave room for the legend on the right of the plot area.

const width = 800, height = 520;
const margin = { top: 70, right: 150, bottom: 60, left: 70 };
const innerW = width  - margin.left - margin.right;
const innerH = height - margin.top  - margin.bottom;

State after step 1: empty SVG with the inner plot area marked out.

Step 2 — load and parse the data

Gapminder is a long table: one row per country per year, from 1952 to 2007. We load the CSV with an accessor (coerces numeric columns), then filter to the 2007 snapshot — 142 rows.

const raw = await d3.csv("./datasets/gapminder.csv", d => ({
    country:   d.country,
    continent: d.continent,
    year:     +d.year,
    lifeExp:  +d.lifeExp,
    pop:      +d.pop,
    gdpPercap:+d.gdpPercap
}));
const data = raw.filter(d => d.year === 2007);

State after step 2: no visible change — the data is in memory, not yet drawn.

Step 3 — define the four scales

One scale per visual channel. Each .domain is computed from the data with d3.extent so the chart adapts to the dataset; each .range is fixed in pixels (or in colors, for the categorical scale).

const x     = d3.scaleLog()   .domain(d3.extent(data, d => d.gdpPercap)).nice().range([0, innerW]);
const y     = d3.scaleLinear().domain(d3.extent(data, d => d.lifeExp)) .nice().range([innerH, 0]);
const r     = d3.scaleSqrt()  .domain(d3.extent(data, d => d.pop))            .range([3, 32]);
const color = d3.scaleOrdinal()
    .domain(["Africa","Americas","Asia","Europe","Oceania"])
    .range (["#009E73","#D55E00","#0072B2","#CC79A7","#E69F00"]);

State after step 3: still no visible change — scales are pure functions, nothing is rendered yet.

Step 4 — canvas and axes

Build the SVG, append the inner plot-area group, add the two axes. The format specifier "~s" renders GDP ticks with SI prefixes (1k, 10k, 100k) instead of long digit strings.

const svg = d3.select("#chart").append("svg")
    .attr("width", width).attr("height", height)
  .append("g")
    .attr("transform", `translate(${margin.left}, ${margin.top})`);

svg.append("g")
    .attr("class", "axis")
    .attr("transform", `translate(0, ${innerH})`)
    .call(d3.axisBottom(x).ticks(6, "~s"));

svg.append("g").attr("class", "axis").call(d3.axisLeft(y));

State after step 4: the two axes appear, but the plot area is still empty of marks.

Step 5 — the data join: one circle per country

Every visual channel comes from the bound row, through the scale defined in step 3:

svg.selectAll("circle")
    .data(data)
    .join("circle")
        .attr("cx",     d => x(d.gdpPercap))     // log-scaled x
        .attr("cy",     d => y(d.lifeExp))       // linear y
        .attr("r",      d => r(d.pop))           // sqrt radius (area ∝ pop)
        .attr("fill",   d => color(d.continent)) // ordinal color
        .attr("opacity", 0.7)
        .attr("stroke",  "white");

Each line of .attr is one visual channel of the scatterplot idiom: position by (x, y), size by area, color by category. This is the moment all five concepts of L9 come together — selection, data, join, scale, attribute.

State after step 5: the scatter is there. Still missing: the legend (no way to know which color is which continent) and the chart title.

Step 6 — title, legend, and tooltips

The legend is built with the same join pattern, but applied to the array of continent names instead of the array of countries. The tooltip is the simplest possible kind: a native SVG <title> element appended to each circle.

// chart title in the outer SVG (above the plot area)
root.append("text")
    .attr("x", margin.left).attr("y", 32)
    .style("font-size", "var(--text-lg)").style("font-weight", "bold")
    .text("Gapminder 2007 — life expectancy vs. GDP per capita");

// tooltip: hover any circle to read the country
svg.selectAll("circle")
    .append("title")
    .text(d => `${d.country}\nlife exp: ${d.lifeExp.toFixed(1)} yr` +
               `\npop: ${d3.format(",")(d.pop)}` +
               `\nGDP pc: ${d3.format("$,.0f")(d.gdpPercap)}`);

// legend: one row per continent, joined to the array of names
const legend = svg.append("g")
    .attr("transform", `translate(${innerW + 20}, 10)`);
const items = legend.selectAll("g.item")
    .data(continents).join("g")
    .attr("class", "item")
    .attr("transform", (d, i) => `translate(0, ${i * 22 + 10})`);
items.append("circle").attr("r", 7).attr("fill", (d, i) => palette[i]);
items.append("text").attr("x", 14).attr("y", 4).text(d => d);

State after step 6: the chart is complete — same as the finished chart at the top of the page.
