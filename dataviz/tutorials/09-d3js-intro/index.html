DV09 — Introduction to D3.js

What you should already know

The tutorials assume:

  • basic HTML — what an element, an attribute, and a class are
  • basic JavaScript — variables, arrow functions, arrays, objects, const / let
  • how to open a file in a web browser

No prior D3, no prior visualization library experience required.

Tutorials

Run them in order — each one builds on the previous.

  1. 09a — Building SVG with D3
    HTML scaffolding, the select · append · attr · style chain, the SVG element catalog (rect, circle, line, path, text, g), the SVG coordinate system, and a live attribute playground. No data yet.
  2. 09b — Data-Driven SVG
    The data join: selectAll · data · join. Toy {x, y, value} dataset placed at literal pixel coordinates (no scales). Explicit visualisation of the enter / update / exit sub-selections, plus a reactive reshuffle demo.
  3. 09c — Scales and Axes
    D3 scales (scaleLinear, scaleLog, scaleBand) on the same dataset with a radio toggle between scatter and bar chart. Margin convention. Axes via d3.axisBottom and d3.axisLeft. Same skeleton applied to a Gapminder slice.
  4. 09d — Loading Data and D3 Helpers
    d3.csv with and without an accessor (string vs number coercion). Helpers (d3.extent, d3.min, d3.max, d3.mean, d3.group, d3.rollup). Formatters (d3.format, d3.timeFormat).
  5. 09e — First Scatterplot
    Capstone: full Gapminder scatterplot end-to-end using everything from the previous tutorials. GDP per capita on a log axis, life expectancy on the y-axis, size by population, color by continent. Legend and tooltips.

Setup

The tutorials are plain HTML files and self-contained — every asset they need (_shared/chart.css, _shared/chart.js, datasets/gapminder.csv) is inside this folder. Serve them with a local HTTP server:

cd 09-d3js-intro
python -m http.server 8000
# open http://localhost:8000/

D3 is loaded from the CDN (https://d3js.org/d3.v7.min.js); no installation required.
