DV09 — Introduction to D3.js

What you should already know

The tutorials assume:

  • basic HTML — what an element, an attribute, and a class are
  • basic JavaScript — variables, arrow functions, arrays, objects, const / let
  • how to open a file in a web browser

No prior D3, no prior visualization library experience required.

Tutorials

Run them in order — each one builds on the previous.

  1. 09a — Building SVG with D3
    HTML scaffolding, the select · append · attr · style chain, the SVG element catalog (rect, circle, line, path, text, g), the SVG coordinate system, and a live attribute playground. No data yet.
  2. 09b — Data-Driven SVG
    The static data join: selectAll · data · join("tag"). Toy {x, y, value} dataset placed at literal pixel coordinates (no scales). What .join does internally — the enter/update/exit decomposition. The dynamic side of the same machinery (key functions and custom callbacks in 10a; transitions in 10b).
  3. 09c — Scales, Axes, and Shape Generators
    D3 scales (scaleLinear, scaleLog, scaleSqrt, scaleBand, scaleOrdinal) on a toy dataset; margin convention; axes via d3.axisBottom / d3.axisLeft. The scale-driven shape generators d3.line and d3.area with curves, .datum() vs .data(), and a ridgeline composition.
  4. 09d — Loading Data and D3 Helpers
    d3.csv with and without an accessor (string vs number coercion). Helpers (d3.extent, d3.min, d3.max, d3.mean, d3.group, d3.rollup). Formatters (d3.format, d3.timeFormat).
  5. 09e — First Scatterplot
    Capstone: full Gapminder scatterplot end-to-end using everything from the previous tutorials. GDP per capita on a log axis, life expectancy on the y-axis, size by population, color by continent. Legend and tooltips.

Setup

The tutorials are plain HTML files and self-contained — every asset they need (_shared/chart.css, _shared/chart.js, datasets/gapminder.csv) is inside this folder. Serve them with a local HTTP server:

cd 09-d3js-intro
python -m http.server 8000
# open http://localhost:8000/

D3 is loaded from the CDN (https://d3js.org/d3.v7.min.js); no installation required.

Per-demo files

The examples/ subfolder contains one self-contained HTML file per demo — each one ships its own copy of styling and scripts, so you can open any of them on its own (or copy a single file into another project) without the rest of the tutorial folder.

Where this leads

L10 picks up the static data join from 09b and turns it into a dynamic one: re-binding with a different array, key functions, transitions, user input, and the layouts (bar, stacked bar, streamgraph) and navigation (d3.zoom) that round out the full advanced D3 vocabulary. L11 / L12 then apply the same machinery to networks.
