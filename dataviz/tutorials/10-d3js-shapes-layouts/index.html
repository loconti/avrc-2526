DV10 — D3 Shapes, Layouts, and Interaction

What you should already know

The L10 tutorials build directly on the L9 sequence. Before starting here, you should be comfortable with:

  • SVG basics — <rect>, <circle>, <text>, attributes, the coordinate system (09a)
  • The data join — selectAll · data · join (09b)
  • Scales and axes — scaleLinear, scaleLog, scaleBand, margin convention (09c)
  • Loading CSVs and helpers — d3.csv, d3.extent, d3.rollup (09d)

Tutorials

Run them in order — each one builds on the previous.

  1. 10a — SVG Paths and the Line / Area Generators
    The SVG path mini-language (M, L, Q, C, Z); reading a path string step by step; d3.line() path generator; .datum() vs .data(); curves (curveLinear · curveStep · curveMonotoneX · …); d3.area(); line + area combined.
  2. 10b — Bar Charts and d3.stack
    Building a bar chart step by step (top vs bottom of bar, the height = innerH - y(value) formula); scaleBand with padding; why stacked bars need transformed data; d3.stack() output structure; nested SVG (g per layer, rect per segment); D3 layouts as a family.
  3. 10c — Update Pattern and selection.join
    Why updates matter; recap of enter/update/exit; the older explicit GUP form (.exit() · .enter() · .merge()); selection.join() as the modern shortcut; key functions vs index joins (interactive demo); custom join with three callbacks.
  4. 10d — Transitions
    selection.transition() basics; transitions after a join; chaining transitions; tuning with duration, delay, ease; easing function comparison (easeLinear · easeCubicInOut · easeBackOut · easeBounceOut · easeElasticOut); staggered transitions via per-datum delay; transitions on enter/exit.
  5. 10e — Interactions Capstone
    selection.on for browser events; the event → state → render loop pattern; hover tooltips; click-to-toggle filters; year-slider scrubbing; full Gapminder interactive scatterplot combining everything from L9 + L10.

Setup

The tutorials are plain HTML files and self-contained — every asset they need (_shared/chart.css, _shared/chart.js, datasets/gapminder.csv) is inside this folder. Serve them with a local HTTP server:

cd 10-d3js-shapes-layouts
python -m http.server 8000
# open http://localhost:8000/

D3 is loaded from the CDN (https://d3js.org/d3.v7.min.js); no installation required.

The examples/ subfolder contains one self-contained HTML file per demo — minimal markup, no narrative text — for live in-class demonstrations. Each file inlines its helpers and can be opened directly in a browser.
